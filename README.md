# CQUPTCrawler

### 重邮学生信息查询和教务在线服务的java库，详细api可见CquptCrawler.java。仅供学习使用。

# How to Use

编译后使用java -jar ${jarname}.jar 即可,编译前需要[commons-tool](https://github.com/wanghaoxin20/commons-tool) 和 [command](https://github.com/wanghaoxin20/command) 库.
